# Amazon Books Scraper

This Python script scrapes the top 10 bestselling books from Amazon.

## Features
- Book Title
- Price
- Ratings (if available)
- Outputs to CSV and JSON

## Technologies Used
- Python
- BeautifulSoup
- requests

## How to Use
1. Clone the repo
2. Run `books_scraper.py`
3. Outputs will be saved in `books.csv` and `books.json`
